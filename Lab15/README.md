# Лабораторная работа №15: Текстовые и бинарные файлы

**Цель:** работа с файлами CSV, бинарными, сжатие RLE.

**Выполнение:**
Чтение CSV через fgets() и strtok(). Вывод таблицы. Сохранение в бинарный файл через fwrite(). Поиск сотрудника по ID через fread().

Сжатие RLE: повторяющиеся байты заменяются на [count][value].

Отчёт: размер до: X Кб, размер после: Y Кб, сжатие: Z%.

**Вывод:** освоена работа с текстовыми и бинарными файлами, сжатие RLE.
