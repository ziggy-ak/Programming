# Лабораторная работа №12: Многофайловый проект. Библиотеки

**Цель:** освоить создание многофайловых проектов и статических/динамических библиотек.

**Выполнение:**

Проект разбит на файлы:
- game.h - заголовочный файл с объявлениями структур и функций
- game.c - реализация функций
- main.c - главная функция
- 1_3.c - дополнительный файл с реализациями (оценка 1-3)

Структура данных (игра):
- name[50] - имя
- level - уровень
- number - номер
- resolution - разрешение

Реализованные функции:
- заполнение массива случайными данными
- вывод таблицы на экран
- сортировка пузырьком

Статическая библиотека (.a):
gcc -c game.c 1_3.c -o game.o
ar rcs libgame.a game.o
gcc main.c -L. -lgame -o program

Динамическая библиотека (.so):
gcc -fPIC -c game.c 1_3.c -o game.o
gcc -shared -o libgame.so game.o
gcc main.c -L. -lgame -o program

**Вывод:** создан многофайловый проект, собраны статическая и динамическая библиотеки.
