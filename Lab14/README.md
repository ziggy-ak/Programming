# Лабораторная работа №14: Связанные списки

**Цель:** реализовать односвязный и двусвязный списки.

**Выполнение:**

Структура игровых данных:
- name[50] - название игры
- genre[50] - жанр
- year - год выпуска
- rating - рейтинг

Структура узла двусвязного списка:
- prev - указатель на предыдущий узел
- data - данные типа game_t
- next - указатель на следующий узел

Реализованные функции:
- NewData() - генерация случайных игр
- NewNode() - создание нового узла
- AddNode() - добавление узла в конец списка
- Bind() - связывание верхнего и нижнего списков
- PrintSpisok() - вывод всего списка
- PrintKey() - навигация по узлам (6 - вперёд, 2 - назад)
- FreeList() - освобождение памяти

Навигация:
- 6 - перемещение к следующему узлу
- 2 - перемещение к предыдущему узлу
- при достижении края списка предлагается вернуться в начало

**Вывод:** реализован двусвязный список с навигацией. Память освобождается корректно.
