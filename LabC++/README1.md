# C++: Paint на SFML

**Цель:** создать графический редактор на C++ с использованием SFML.

**Выполнение:**
Реализованы инструменты: кисть, прямоугольник, линия, ластик.

Функции:
drawPixel() - рисование пикселя
drawBrush() - круглая кисть
drawRectangle() - прямоугольник
drawLine() - линия (алгоритм Брезенхема)
undo() - отмена действий

Фильтры: оттенки серого, негатив, размытие 3x3.

Выбор цвета: по цифрам 0-9 или кликом по палитре.

Управление:
B - кисть, R - прямоугольник, L - линия, E - ластик
+/- - размер кисти
C - очистить, Ctrl+Z - отменить
G - оттенки серого, N - негатив, M - размытие

**Вывод:** создан графический редактор Paint с основными инструментами и фильтрами.
