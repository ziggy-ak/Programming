# Лабораторная работа: Paint на C++ с использованием SFML

Основная задача - дополнить недостающие функции, освоив базовые конструкции языка  C++, алгоритмы и контейнеры STL, а также работу с событиями мыши и клавиатуры на примере простой библиотеки SFML.

## Окружение:

- Система сборки CMake (версия 3.10+).
- Компилятор C++17 (g++, clang++).
- Библиотека [SFML](https://www.sfml-dev.org/) версии 2.5 или новее:
  - **Ubuntu/Debian:** sudo apt install libsfml-dev
  - **macOS:** brew install sfml
  - **Windows (vcpkg):** vcpkg install sfml:x64-windows
  - **Windows (manual):** Download SFML and set -DSFML_DIR=C:/SFML/lib/cmake/SFML

### Запуск графики в WSL

WSL не имеет собственного графического интерфейса, поэтому потребуется использовать X-сервер (например, VcXsrv или WSLg).

**Для WSL 2 с WSLg (встроенная поддержка GUI):**
- Обновите WSL до версии с поддержкой WSLg (Windows 10/11). Графика будет работать автоматически, дополнительно ставить ничего не нужно

**Для WSL 2 с внешним X-сервером (VcXsrv):**
1. Установите VcXsrv на Windows (https://sourceforge.net/projects/vcxsrv/).
2. Запустите XLaunch, выберите "Multiple windows", Display number = 0, снимите галочку "Native opengl", отметьте "Disable access control".
3. В WSL выполните:
```bash
   export DISPLAY=$(cat /etc/resolv.conf | grep nameserver | awk '{print $2}'):0
```
или добавьте эту строку в конец `~/.bashrc` и пропишите `source ~/.bashrc`

Проверка работоспособности
Скомпилируйте и запустите. Если появляется окошко, то все настроено верно

## Сборка и запуск проекта
Собираем:
```bash
cd project          # перейдите в корневую папку проекта
mkdir build && cd build
cmake ..
make
```
Запускаем:
```bash
./paint
```

## Структура проекта
```
paint++/
├── CMakeLists.txt
└── src/
    ├── paint_core.hpp
```

## Подсказка по ф-иям

- **`canvas`** - холст/таблица пикселей (двумерный вектор цветов)
- **`WIDTH / HEIGHT`** - размер холста в пикселях (200×150)
- **`PIXEL_SIZE`** - масштаб пикселя на экране
- **`undoStack`** - стек для отмены действий
- **`sf::Color`** - цвет (r, g, b)
- **`drawPixel`** - поставить одну точку
- **`drawBrush`** - нарисовать круг (кисть/ластик)
- **`drawRectangle`** - закрашенный прямоугольник
- **`drawLine`** - отрезок (алгоритм Брезенхема) ([prog-cpp.ru/brezenham](https://prog-cpp.ru/brezenham/))
- **`saveToUndo`** - сохранить холст (canvas) в стек
- **`undo`** - восстановить из стека
- **`applyGrayscale`** - обесцветить (оттенки серого)
- **`applyNegative`** - инвертировать цвета
- **`applyBlur`** - размыть (усреднение 3×3)
